Leaning to add links to HTML


Use "a" tag to create a link and write the name you want to display, in the tag


The properties of the link are listed inside the <> of the "a" tag. The properties that can be used are:-

1.href:

Type href="" and write the link inside the quotes

2.target:

Type target=_blank to open the link in a new page or target=_self to open the link in the same page

3.title:

Type title="" to write the text that will pop up when the mouse is hovering over the link


You can also use the link to open a file that is currently opened in vs code by writing its name in the href tag

You can also use href to mail some by using "mailto:" inside the href and writing the mail after the colon
