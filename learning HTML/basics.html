Text Formatting

different tags can be used around a word or sentence to change its formats. The tags that can be used are:-

1.b can be used to write bold letters

2.i can be used to write italic letters

3.big can be used to write big letters

4.small can be used to write small letters

5.sub can be used to write subscript letters

6.sup can be used to write superscript letters

7.ins can be used to underline letters

8.del can be used to write letters that are crossed by a line

9.mark can be used to write letters that are highlighted
