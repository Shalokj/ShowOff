Leaning to add images to HTML


Use "img" inside <> to create an image tag


The source and properties of an image are all written together inside the <>

src="" is used to get the image from the source by writing the name or path of the image inside ""

The properties of an image that can be changed are:-

1.Height:

The height of an image can be changed by using height="" and writing a number inside""

2.Width

The width of an image can be changed by using width="" and writing a number inside ""

3.Alt:

This can be used to provide a description of an image if a screen reader is being used

4.Title

Type title="" to write the text that will pop up when the mouse is hovering over the image


An image can be converted to a link that will open when the image is clicked by surrounded the img tag in the "a" tag
